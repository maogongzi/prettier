#### Treat `.prettierrc` as YAML when formatting it (#8105 by @fisker)

The `.prettierrc` file can be written in either JSON or YAML. Previously, when Prettier formatted it, the parser was inferred to be `json`, which lead to a `SyntaxError` thrown if the content was YAML. Now itâ€™s treated as a YAML file. However, if it's JSON, it will be formatted as JSON (not as JSON-like YAML).
